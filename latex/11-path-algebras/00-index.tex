\chapter{Chapter 11: Quivers and path algebras}\label{sec:11-path-algebras:00-index:1}

This chapter combines everything covered so far: inductive types (Chapter 1), structures (Chapter 2), and rings (Chapters 7--8), used to build a genuinely interesting example: the \textbf{path algebra} of a quiver.

\textbf{Learning objectives.} By the end of this chapter, encode a quiver as a \texttt{structure} of vertices/arrows/source/target, define \texttt{Path} as an inductive type indexed by its own endpoints, compose paths via \texttt{Path.append} and predict when the composability side-conditions block a nonsensical composition, and explain in what sense a path algebra \(kQ\) extends the \texttt{Ring}/\texttt{Module} machinery of Chapters 8 and 10.
