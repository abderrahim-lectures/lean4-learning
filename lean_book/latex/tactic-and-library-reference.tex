\addcontentsline{toc}{chapter}{Tactic and library reference}

A quick index of every tactic used in this book, and every Mathlib name used in the ``Mathlib equivalent'' boxes (Chapters 6-11), each with a link to look it up yourself. This page is a lookup table, not something to read start to finish --- the tactics chapter (\hyperref[sec:04-tactics:00-index:1]{Chapter 4}) and the working-efficiently chapter (\hyperref[sec:12-working-efficiently:00-index:1]{Chapter 12}) are where each one is actually explained.

Two general links used throughout this page:

\begin{itemize}
\tightlist
\item
  \textbf{\href{https://lean-lang.org/doc/reference/latest/Tactic-Proofs/Tactic-Reference/}{Lean 4 Tactic Reference}} --- the official, current documentation for every built-in tactic.
\item
  \textbf{\href{https://loogle.lean-lang.org/}{Loogle}} --- the Mathlib declaration search engine; \texttt{https://loogle.lean-lang.org/?q=NAME} jumps straight to a name. Also see the browsable \textbf{\href{https://leanprover-community.github.io/mathlib4_docs/}{Mathlib4 docs}}.
\end{itemize}
